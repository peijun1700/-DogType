## Global Art Direction

### 核心風格

- `Soft Character Card`
- 奶白人格卡 / 柔和陰影 / 低飽和配色
- 幾何低多邊形角色 / 測驗結果卡 / 輕產品感
- 不是硬核 dashboard，也不是貼圖狗，而是「可被辨識的角色型人格犬」

### 統一世界觀

- 每隻狗都是一張「舔狗人格測驗卡」中的主角角色
- 角色本身有名字、代號、人格標籤
- 畫面主體是狗，不是 UI
- 視覺必須白、輕、乾淨、可分享

### 共通畫面元素

- 奶白或極淡薄荷背景
- 柔霧光、極淡投影、卡片式留白
- 畫面中固定有：
  - 中文狗種名
  - 英文代號
  - 中央角色
  - 一句人格評語

### 共通材質

- 主材質：霧面紙卡 + 柔塑角色表面
- 角色表面可做低多邊形 3D 或類 3D 幾何切面
- 反光要弱，偏柔和，不要潮流 3D 鏡面質感

### 共通構圖

- 角色居中
- 上方是中文名與英文代號
- 中間是角色
- 下方是一句人格台詞
- 像心理測驗結果卡，不像產品規格表

## Character Card Prompt Structure

每張卡建議固定長這樣：

1. `你的舔狗品種是：`
2. 中文狗種名
3. 英文代號
4. 中央低多邊形角色
5. 一句人格台詞

## Shared Prompt Template

可先用這個統一模板，再替換品種與特徵：

```text
A soft personality test card, low poly dog character, matte pastel surface, clean white card layout, centered composition, soft diffused studio lighting, minimal shadow, pale mint background, cute but refined, premium visual identity, Chinese title, English codename, shareable result card aesthetic
```

## Per-Breed Prompt Formula

```text
[breed name], [codename], [personality keywords], [accessory keywords], low poly dog character, clean white personality card, matte pastel surface, centered composition, soft diffused studio lighting, minimal shadow, pale mint background, premium cute visual identity
```

## Negative Prompt Suggestions

```text
sticker pack, chibi mascot, neon cyberpunk, glossy toy plastic, messy background, highly saturated colors, cheap mobile game art, childish proportions, horror gore, dark dramatic lighting
```

## Usage Recommendation

第一階段建議先做兩層素材：

1. `Character Portrait`
   - 每種狗一張主視覺
   - 用於結果卡與分享圖

2. `Card Insert Variant`
   - 同角色但更小、更簡潔
   - 用於小卡、列表、未來養成 UI

如果之後要進入遊戲化，再補：

3. `Idle / Happy / Broken / Alert` 四種狀態圖
4. 小型 icon 版頭像
