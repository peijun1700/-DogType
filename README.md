# 🦴 SoulSync: DogType Analyzer (舔狗類型鑑定器)

SoulSync 是一款基於互動拓撲（Interaction Topology）的關係診斷工具。它能透過解析 LINE 的聊天紀錄，利用 17 層個性化演算法，鑑定出你在這段關係中扮演的「犬種人格」。

---

## ✨ 核心特色

- **13 種精確犬種人格**：從「高冷黑貓」到「哈士奇」，我們利用 Master Gate 邏輯進行權力地位分流。
- **關係模型判定**：鑑定你們是「雙向同步」、「單向輸出」還是「間歇性強化」。
- **100% 隱私保護**：純前端 JavaScript 解析，對話紀錄**不會離開**你的瀏覽器，安全無虞。
- **IG 限動分享**：自動生成設計感十足的人格卡片，方便一鍵分享。

## 🐕 基因圖鑑預覽

- **高冷黑貓型**：高位、被動、低輸出，維持優雅的距離感。
- **佛系和尚狗型**：心如止水，情緒穩定如磐石。
- **狼系主宰型**：掌握絕對話語權，引導對方的呼吸節奏。
- **貴賓狗型**：精心打理的回覆，體面且富有包裝感。
- **殭屍狗型**：連線已斷開，但你的心跳包（訊息）還在單向輸出。

## 🛠️ 技術底層

本專案通過了 **Omega-27 工業級架構認證** 與 **17 層代碼質量核驗**，具備極高的健壯性與數據準確度。

- **指標**：Latency (延遲), Payload (內容投入), Jitter (忽冷忽热), Penalty (危險訊號)。
- **引擎**：核心判定逻辑基於 `computeDogSignals` 非對稱地位演算法。
